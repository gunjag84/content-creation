## What Was Built

### Task 1: PresetsService + IPC + Preload

- **`SlidePreset` type** added to `src/shared/types/generation.ts` - id, name, template_id, zone_overrides, overlay_opacity, created_at
- **`PresetsService`** (`src/main/services/presets-service.ts`) - synchronous JSON read/write with loadPresets, savePreset (upsert by id), deletePreset
- **IPC handlers** (`src/main/ipc/presets.ts`) - `presets:list`, `presets:save`, `presets:delete`
- **Registered** in `src/main/index.ts` via `registerPresetsHandlers()`
- **Preload API** extended in `src/preload/types.ts` and `src/preload/index.ts` with `presets: { list, save, delete }`
- **5 round-trip TDD tests** covering: empty state, save+load, upsert, delete, overlay_opacity persistence

### Task 2: applyPreset Store Action

- `applyPreset(slideIndex, preset)` action added to `useCreatePostStore`
- Sets `zone_overrides` from preset (full replacement, not merge)
- Applies `overlay_opacity` only if defined on preset
- Pushes previous slides to `slideHistory` (undo stack, capped at 50)
- Clears `slideHistoryFuture` on apply
- **4 TDD tests** covering all behaviors, undo works after applyPreset
