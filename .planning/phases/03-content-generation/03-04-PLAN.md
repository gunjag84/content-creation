  <behavior>
    Generation IPC channels:

    'generate:content' (POST-06):
    - Args: { prompt: string }
    - Loads API key via loadApiKey() - returns error if null/empty
    - Creates Anthropic client with key
    - Streams via client.messages.stream (model: 'claude-sonnet-4-5-20250929', max_tokens: 4096)
    - On each text event: win.webContents.send('generate:token', token)
    - On complete: parse finalMessage().content[0].text as JSON -> GenerationResult
    - win.webContents.send('generate:complete', generationResult)
    - On error: win.webContents.send('generate:error', { message: err.message })
    - Returns { started: true } immediately (streaming continues async)

    'generate:hooks' (POST-09):
    - Args: { currentHook: string, slideContext: string, prompt: string }
    - Same streaming pattern but asks for 3 hook alternatives
    - On complete: win.webContents.send('generate:hooks-complete', { hooks: string[] })

    'generate:stories' (STORY-01 through STORY-07):
    - Args: { prompt: string } (assembleStoryPrompt handles building)
    - Streams story proposals
    - On complete: parse as StoryProposal[] array (2-4 proposals)
    - win.webContents.send('generate:stories-complete', { proposals: StoryProposal[] })

    Export IPC channels (POST-16, STORY-10):

    'export:select-folder':
    - Uses dialog.showOpenDialog with properties: ['openDirectory', 'createDirectory']
    - Returns { canceled: boolean, path?: string }

    'export:save-files' (POST-16):
    - Args: { folderPath: string, files: ExportFile[] }
    - For .png files: decode base64 data URL (strip 'data:image/png;base64,'), Buffer.from(base64, 'base64'), fs.writeFile(path, buffer)
    - For .txt files: fs.writeFile(path, content, 'utf-8')
    - Returns { success: boolean, error?: string }
    - Disable double-export: track 'exporting' flag in IPC handler state, return error if already in progress

    Preload additions in preload/index.ts and preload/types.ts:
    - generation.streamContent(prompt: string): Promise<{ started: boolean }>
    - generation.streamHooks(args): Promise<{ started: boolean }>
    - generation.streamStories(prompt: string): Promise<{ started: boolean }>
    - generation.onToken(cb: (token: string) => void): () => void  // returns cleanup fn
    - generation.onComplete(cb: (result: GenerationResult) => void): () => void
    - generation.onHooksComplete(cb: (result: { hooks: string[] }) => void): () => void
    - generation.onStoriesComplete(cb: (result: { proposals: StoryProposal[] }) => void): () => void
    - generation.onError(cb: (error: { message: string }) => void): () => void
    - export.selectFolder(): Promise<{ canceled: boolean; path?: string }>
    - export.saveFiles(folderPath: string, files: ExportFile[]): Promise<{ success: boolean; error?: string }>
