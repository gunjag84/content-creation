- Card with "Balance Overview" heading
- Body: "No posts yet - create your first post to see balance insights."

**Warm state layout (total_posts > 0):**

Section 1 - Content Pillars (LEARN-06):
- Title: "Content Pillars" + total post count badge
- For each pillar: two horizontal bars stacked
  - Top bar (actual, bg-blue-500): width = actual_pct%
  - Bottom bar (target, border border-blue-500/40 dashed): width = target_pct%
  - Label: "{pillarName} ({count} posts, {actual_pct.toFixed(0)}% actual / {target_pct}% target)"
  - If |actual_pct - target_pct| > 15: show amber dot (w-2 h-2 rounded-full bg-amber-500 inline-block)

- Title: "Mechanics"

- Bar width = (count / max_count) * 100%
- If mechanic name appears in any warning: amber dot before label

Section 3 - Themes:
- Title: "Themes" + note if > 8 themes visible ("showing top 8")
- Show top 8 by count, same bar pattern as mechanics

Section 4 - Rotation Alerts (only if warnings.length > 0):
- Collapsible "Rotation Alerts" section (open by default if any warnings)
- Each warning: amber AlertTriangle icon + message text
