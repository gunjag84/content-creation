# Content Dimensions - LEBEN.LIEBEN

MECE taxonomy for Instagram content creation. Every post is tagged with one value per required dimension. The combination engine uses these to ensure variety across the feed.

## Overview

| Dimension | Code | Required | Count | Purpose |
|-----------|------|----------|-------|---------|
| Pillar | P1-P3 | Yes | 3 | What business goal does this post serve? |
| Area | L1-L7 | Per pillar | 7 | What life area does this post cover? |
| Angle | per pillar | Yes | 11 total | What sub-angle within the pillar? |
| Method | M1-M13 | Yes | 13 | How is the post structured? |
| Tonality | T1-T5 | Yes | 5 | What emotional register? |

## Pillars

Business goal the post serves. Each pillar has its own rules, allowed dimensions, and angles.

| ID | Name | Target % | Purpose |
|----|------|----------|---------|
| P1 | Generate Demand | 65% | Awareness, reach, new followers. Pain + aspiration content. |
| P2 | Convert Demand | 25% | Drive purchases. Product, proof, ritual content. |
| P3 | Nurture Loyalty | 10% | Retain buyers. Community, behind-the-scenes, brand affinity. |

### Pillar Rules

**Generate Demand (65%)**

Ziel: Aufmerksamkeit, Wiedererkennung, emotionale Resonanz. Kein Produkt, kein Pitch. Die Leserin soll denken: "Das bin ich. Die versteht mich."

- Sprich das Gefühl an, nie die Lösung. Zeige das Problem in Alltagsszenen, die Luise sofort wiedererkennt.
- Nutze Kontraste: voller Kalender vs. leeres Herz. Funktionieren vs. Fühlen.
- Wissenschaft nur als überraschenden Fakt einstreuen (40%-Statistik, Negativity Bias), nie dozierend.
- Humor darf schärfer sein als im Rest - augenzwinkernd über den Wahnsinn des Alltags.
- Jeder Post muss teilbar sein: "Tagge eine Freundin" oder Screenshot-würdig.
- Kein Journal, kein Produkt, kein Link, kein Shop. Reine Resonanz.

**Convert Demand (25%)**

Ziel: Interesse in Kaufabsicht wandeln. Produkt zeigen, Nutzen erklären, Einladung formulieren.

- Immer konkreten Nutzen vor Feature: Was verändert sich im Alltag, nicht was das Journal enthält.
- WEIL3-Methode erklären und demonstrieren - sie ist der Differentiator zum 6-Minuten-Tagebuch.
- Social Proof mit echten Geschichten, nie mit generischen Testimonials.
- Wissenschaft als Vertrauensanker: Emmons-Studie, Lyubomirsky 40%, Harvard Grant Study.
- CTA als Einladung: "Wenn Du magst, findest Du das Journal im Shop." Nie als Druck.
- Preis darf genannt werden (34,90 EUR / 29,90 EUR Launch). Analog-Argument gegen App-Konkurrenz.

**Nurture Loyalty (10%)**

Ziel: Beziehung vertiefen, Community stärken, Markenbindung. Kein Produkt, kein Pitch.

- Jules echte Geschichte erzählen: die Krise, die Erkenntnis, der Weg. Verletzlich, nicht poliert.
- Behind the Scenes zeigt Entscheidungen und Werte, nicht nur hübsche Bilder.
- Community aktiv einbinden: Fragen stellen, Challenges starten, Nutzerinnen-Wins feiern.
- Thought Leadership über die Mission: analog vs. digital, Tiefe vs. Breite, Erlaubnis vs. Optimierung.
- Kein Produkt-Push. Kein CTA zum Shop. Die Community soll sich zugehörig fühlen, nicht verkauft.

## Areas (Lebensbereiche)

The "what" of each post. Which slice of the audience's life are we entering?

| ID | Name | Description |
|----|------|-------------|
| L1 | Familie & Beziehungen | Partner, Kinder, Miteinander, Distanz, Verbindung. Das Zwischenmenschliche im Familienleben. |
| L2 | Innere Welt | Mental Load, Schuldgefühle, Selbstzweifel, Rollenkonflikte, verschüttete Leidenschaften. "Sollte das mein Leben sein?" |
| L3 | Alltagslogistik | Zeitknappheit, Multitasking, Pendelzeit, unsichtbare Arbeit. Der Wettlauf gegen die Uhr. |
| L4 | Digitale Welt | Smartphone-Sucht, Scrollen, Messenger-Chaos, always-on. Immer erreichbar, nie abgeschaltet. |
| L5 | Produkt & Angebot | Das Journal, Unboxing, Anwendung, Bundle, Geschenkidee. Conversion-Kern. |
| L6 | Marke & Gründerin | Jules Geschichte, Behind the Scenes, Produktentwicklung, Nachhaltigkeit. |
| L7 | Community & Social Proof | Testimonials, UGC, Challenges, Nutzerinnen-Geschichten. Vertrauen durch echte Stimmen. |

## Methods (Methoden)

The storytelling structure. How is the post built?

| ID | Name | Description | Format |
|----|------|-------------|--------|
| M1 | Provokante These | Starke, polarisierende Aussage als Hook. Kontrovers, stoppt den Scroll. | Both |
| M2 | Frage & Antwort | Frage als Hook die zum Nachdenken zwingt, Antwort liefert den Mehrwert. | Both |
| M3 | Persönliche Geschichte | Storytelling. Szene, Gefühl, Erkenntnis. | Both |
| M4 | Vorher / Nachher | Transformation zeigen. Kontrast zwischen Problem und Lösung. | Both |
| M5 | Testimonial | Echte Nutzerinnen-Geschichten, UGC, Rezensionen. Social Proof. | Both |
| M6 | Zitat & Kommentar | Starkes Zitat als Aufhänger, persönlicher Kommentar als Einordnung. | Single only |
| M7 | Faktencheck | Wissenschaftlicher Fakt + überraschender Twist. Daten schaffen Glaubwürdigkeit. | Both |
| M8 | Liste | Listicle: 3/5/7 Dinge die... Leicht konsumierbar, hoher Save-Wert. | Both |
| M9 | Mythos vs. Realitat | Verbreitete Annahme widerlegen. Überraschungseffekt. | Both |
| M10 | Anleitung | Schritt-für-Schritt How-to. Praktischer Mehrwert, sofort umsetzbar. | Carousel only |
| M11 | Call to Action | Direkter Conversion-Post. Produkt zeigen, Nutzen erklären, einladen. | Single only |
| M12 | Behind the Scenes | Blick hinter die Kulissen. Produktentwicklung, Material, Entscheidungen. | Both |
| M13 | Beobachtung | Alltagsbeobachtung als Aufhänger. Etwas Kleines gesehen, größere Erkenntnis. | Both |

## Tonalities (Tonalitat)

The emotional register of the post.

| ID | Name | Description | When to use |
|----|------|-------------|-------------|
| T1 | Emotional | Warm, berührend, verletzlich. Herz über Kopf. | Family moments, personal breakthroughs, vulnerability |
| T2 | Humorvoll | Leicht, witzig, augenzwinkernd. Alltag mit Humor. | Relatable chaos, everyday absurdity, scroll-stoppers |
| T3 | Sachlich | Faktenbasiert, ruhig, informativ. Daten sprechen lassen. | Science posts, statistics, method explanations |
| T4 | Provokant | Kantig, herausfordernd, unbequem. Reibung erzeugen. | Myth-busting, hot takes, challenging assumptions |
| T5 | Ermutigend | Aufbauend, stärkend, du schaffst das. Energie geben. | Motivation, self-worth, community building |

**Note:** Tonality applies to the body text and caption. Hooks follow their own rules and may be sharper than the post's overall tonality (per content strategy: "Hook-Regeln folgen NICHT der Brand Voice - dürfen schärfer sein").

## Constraint Matrix

When a pillar is selected, only certain dimensions are available. This is enforced in the wizard UI and the generation engine.

### Tonalities per Pillar

|  | T1 Emotional | T2 Humorvoll | T3 Sachlich | T4 Provokant | T5 Ermutigend |
|--|:---:|:---:|:---:|:---:|:---:|
| **Generate Demand** | x | x | x | x | x |
| **Convert Demand** | x | - | x | - | x |
| **Nurture Loyalty** | x | - | - | - | x |

### Methods per Pillar

|  | M1 | M2 | M3 | M4 | M5 | M6 | M7 | M8 | M9 | M10 | M11 | M12 | M13 |
|--|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **Generate Demand** | x | x | x | x | - | x | x | x | x | - | - | - | x |
| **Convert Demand** | - | x | x | x | x | - | x | x | x | x | x | - | - |
| **Nurture Loyalty** | x | x | x | - | - | x | - | - | - | - | - | x | x |

Logic:
- **Generate Demand** gets all awareness/emotion methods. No direct selling (M5, M10, M11) and no BTS (M12).
- **Convert Demand** gets all product/proof methods. No provocative-only (M1), no pure brand methods (M6, M12, M13).
- **Nurture Loyalty** gets relationship-building methods. No selling (M5, M10, M11), no data-heavy (M7, M8, M9), no transformation framing (M4).

### Areas per Pillar

|  | L1 Familie | L2 Innere Welt | L3 Alltag | L4 Digital | L5 Produkt | L6 Marke | L7 Community |
|--|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **Generate Demand** | x | x | x | x | - | - | - |
| **Convert Demand** | x | x | x | x | x | - | x |
| **Nurture Loyalty** | - | - | - | - | - | x | x |

Logic:
- **Generate Demand** uses life areas (L1-L4) where the pain/aspiration lives. No product, no brand.
- **Convert Demand** contextualizes the product (L5) within life areas (L1-L4) and adds social proof (L7).
- **Nurture Loyalty** is brand-only (L6) and community (L7). Area is optional for this pillar.
